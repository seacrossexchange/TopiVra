# feature-delivery

## Workflow
1) Clarify acceptance criteria (AC) + non-goals
2) Identify impacted modules (FE/BE/DB)
3) Plan: tasks + sequencing + rollback
4) Implement with small commits
5) Tests: unit + integration + smoke
6) Docs: update API/docs and add release note
7) Output: PR summary + AC checklist + how to verify
